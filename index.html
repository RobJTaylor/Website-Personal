Espresso


Espresso icon

Espresso is a little Windows application I wrote a few years back.
It's similar to the 'caffeine' app for macOS - it keeps the computer awake.
I wrote it using XAML islands, which enabled modern Windows UI controls using WPF.

It is available in the Microsoft Store for free download.

Mon API


Mon API icon

'Mon API' (temporary name) is a work-in-progress Android/iOS app.
It's powered by React Native, and acts as visual method of interacting with the Pokemon API.
Data is persisted using redux-persist, so data only needs to be fetched once.
Pokemon are also searcahble using a 'like' query.

It is available in the Play Store for free download.
